# Install

```
npm install https://github.com/subhero24/tailwindcss-3d
```

Add the plugin to your tailwind config file

```javascript
import tailwind3dPlugin from "tailwindcss-3d";

export default {
	plugins: [tailwind3dPlugin],
};
```

# Usage

.3d
.flat

.backface-hidden
